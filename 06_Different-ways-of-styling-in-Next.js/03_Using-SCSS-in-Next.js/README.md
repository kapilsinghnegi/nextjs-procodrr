# Using SCSS in Next.js

Sass is a CSS preprocessor with some extra features. It has 2 syntaxes: SCSS(similar to CSS) and Sass. In companies mainly SCSS is used. To use Sass in Next.js we need to install `sass` package - `npm install --save-dev sass`.

Every css code is a valid Sass code.

We can also ue scss file as module CSS file.

We can customize the output of the css file using `sassOptions` property in next.config.js.
