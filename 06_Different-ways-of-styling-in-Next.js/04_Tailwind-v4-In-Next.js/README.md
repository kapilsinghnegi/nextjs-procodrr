# Tailwind CSS in Next.js

To setup Tailwind CSS in new Next.js project is very easy, we need to run `npx create-next-app@latest my-app --tailwind` command or just `npx create-next-app@latest my-app` command and choose `Yes` for Tailwind CSS.

To use it in an existing project, we can follow Tailwind CSS installation guide [here](https://tailwindcss.com/docs/installation/framework-guides/nextjs).
